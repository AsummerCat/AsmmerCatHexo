## 换行
````
在文字结尾加上两个空格一个回车 自动换行
文字之间间隔一行 表示换段
````

## 列表
### 无序列表
```
无序列表就是前面加个小圆点
语法: 
三种写法，使用 * 或 + 或 - ：

使用 * ：
* TEST1
* TEST2
* TEST3

使用 + ：
+ TEST4
+ TEST5
+ TEST6

使用 - ：
- TEST7
- TEST8
- TEST9
```
**呈现效果:**  

* TEST1
* TEST2
* TEST3

**当然还可以嵌套使用:**  

````
* 上一级    
  * 下一级1
  * 下一级2
````

* 上一级    
  * 下一级1
  * 下一级2

---

### 有序列表
```
语法: 有序列表就是前面加个序列号  (哈哈 还真的... 通俗易懂)
tips:如果排序错误不影响 默认升序
例:
  1. 小明
  2. 小东
  3. 小杰
  1. 小嘿
  3. 小模
  2. 小😯
  
两个效果等同
```

  1. 小明
  2. 小东
