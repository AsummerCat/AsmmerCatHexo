# 1.聚类算法

使用不同的聚类种子,产生的聚类结果不一样
`聚类算法是是一种无监督学习算法,相似的样本自动归到一个类别下`

*   聚类算法分类
    粗聚类
    细聚类

*   聚类算法定义
    1.是一种无监督学习算法
    2.主要用于将样本自动归到一个类别下
    3.计算样本之间的相似性,一般使用欧式距离

# 2.聚类算法和分类算法的区别

`聚类算法`是无监督学习算法
`分类算法`是监督学习算法



# 3.聚类算法API的使用

```
sklean.cluster.KMeans(n_cluster=8)

参数: 
 * n_cluster: 开始聚类数量 (相当于 你需要分为多少个类别)
  默认为8, 生成的聚类数,即产生的质心(centroids)数
  
方法: 
fit(X[, y]): 训练模型。
predict(X): 预测给定数据的最近的簇中心。
fit_predict(X[, y]): 计算聚类中心，并返回每个样本所属的簇标签。
transform(X): 将数据转换为簇距离空间。
fit_transform(X[, y]): 对训练数据进行拟合并转换，返回转换后的数据。

```

#### 3.1 案例 KMeans

*   `calinski_harabasz_score` 评估聚类效果(CH指数)

`calinski_harabasz_score` 用来评估聚类效果 指数的值越大，通常意味着聚类的效果越好。
使用:

```
from sklearn.metrics import calinski_harabasz_score
   
# 获取聚类标签
labels = kmeans.labels_
score = calinski_harabasz_score(X, labels)

```

*   `KMeans` API案例

<!---->

    from sklearn.cluster import KMeans
    from sklearn.datasets import make_blobs
    from sklearn.metrics import calinski_harabasz_score
    import matplotlib.pyplot as plt

    # 生成模拟数据
    X, _ = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=0)

    # 定义一个范围来测试不同的聚类数
    range_n_clusters = [2, 3, 4, 5, 6]

    # 存储每种聚类数下的 Calinski-Harabasz 分数
    ch_scores = []

    for n_clusters in range_n_clusters:
        # 实例化 KMeans 模型
        kmeans = KMeans(n_clusters=n_clusters, random_state=0)
        
        # 拟合模型
        kmeans.fit(X)
        
        # 获取聚类标签
        labels = kmeans.labels_
        
        # 计算 Calinski-Harabasz 分数
        score = calinski_harabasz_score(X, labels)
        ch_scores.append(score)
        
        print(f"Number of Clusters: {n_clusters}, Calinski-Harabasz Score: {score}")

    # 绘制不同聚类数下的 Calinski-Harabasz 分数
    plt.figure(figsize=(10, 6))
    plt.plot(range_n_clusters, ch_scores, marker='o')
    plt.title('Calinski-Harabasz Score vs Number of Clusters')
    plt.xlabel('Number of Clusters')
    plt.ylabel('Calinski-Harabasz Score')
    plt.grid(True)
    plt.show()

# 4.算法优化指标

*   sse : 误差平方的值越小越好
*   肘部法: 下降率突然变缓的点,即认为是最佳的k值
*   SC系数: 取值在\[-1,1]之间,越大越好
*   CH系数: 分数越高越好,注意需要:尽可能少的类别聚类尽量多的样本,并且保持较好的聚类效果

# 5.算法优化

还有一些常见的聚类算法
`kmean`,`kmean++`,`二分k-means`,`k-medoids`

*   `canpoy` 预处理聚类算法，常用于减少 K-Means 等算法的计算量

#### 5.1 canpoy算法的使用

通过快速筛选出可能属于同一簇的样本，减少后续聚类算法的计算负担。
`计算速度快，适合大规模数据集的预处理`。
`可以显著减少后续聚类算法的计算量`。
PS: 注意该方法只能用来`预处理`,不能生成最终结果
工作原理: 类似于 不断重复在圆内确定各个点

#### 5.2 K-Mean++ 基于kmean的改良版本

通过更合理的初始质心选择，提高 K-Means 的聚类效果
目的:`让选择的质心更加的分散`

    from sklearn.cluster import KMeans
    from sklearn.datasets import make_blobs
    import matplotlib.pyplot as plt

    # 生成模拟数据
    X, _ = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=0)

    # 实例化 KMeans++ 模型
    kmeans_plus_plus = KMeans(n_clusters=4, init='k-means++', random_state=0)

    # 拟合模型
    kmeans_plus_plus.fit(X)

    # 获取聚类标签
    labels = kmeans_plus_plus.labels_

    # 获取聚类中心
    centers = kmeans_plus_plus.cluster_centers_

    # 绘制数据点及其对应的聚类中心
    plt.scatter(X[:, 0], X[:, 1], c=labels, s=50, cmap='viridis')
    plt.scatter(centers[:, 0], centers[:, 1], c='red', s=200, alpha=0.75)
    plt.show()

#### 5.3 KMedoids 基于划分的聚类算法。

类似于 K-Means，但使用簇内的实际样本点作为质心

    from sklearn_extra.cluster import KMedoids
    from sklearn.datasets import make_blobs
    import matplotlib.pyplot as plt

    # 生成模拟数据
    X, _ = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=0)

    # 获取聚类标签

    # 获取聚类中心

    # 绘制数据点及其对应的聚类中心

    plt.scatter(centers[:, 0], centers[:, 1], c='red', s=200, alpha=0.75)
    plt.show()
