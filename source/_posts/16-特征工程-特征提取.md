# 特征工程-特征提取

## 1.特征提取

将文字/图像/文字类别 之类的非数值特征 转换为(`sparse矩阵`) 方便机器学习识别
`将任意数据(如文本或图像)`转换为可用于机器学习的数字特征

*   特征提取分类
    \*\* 字典(类别)特征提取(特征离散化)
    \*\* 文本特征提取
    \*\* 图像特征提取( 深度学习)

## 2.特征提取API

    sklearn.feature_extraction

*   这里分为单一特征提取和多特征提取
    单一特征提取: LabelEncoder
    多特征提取: DictVectorizer

#### 2.1 字典特征提取 DictVectorizer

*   API

<!---->

    from sklearn.feature_extraction import DictVectorizer

    调用fit_transform(X)进行输入和转换

    X:表示字典或者包含字典的迭代器返回值

    入参:
     * sparse来控制是否返回矩阵
       sparse=False

    返回参数:
    DictVectorizer.get_feature_names() 返回类别名称

*   案例

<!---->

    from sklearn.feature_extraction import DictVectorizer
    # 特征提取-字典提取

    ## 特征数据集
    x=[{'city': '北京','temperature':100},
    {'city': '上海','temperature':60},
    {'city': '深圳','temperature':30}]

    ## 实例化字典提取器 sparse来控制是否返回矩阵
    ver=DictVectorizer(sparse=False)
    ## 调用fit_transform进行输入和转换
    res=ver.fit_transform(x)
    ## 返回特征名称
    print('特征名称:',ver.feature_names_)
    ## 返回矩阵
    print('返回结果:\n',res)

#### 2.2 文本特征提取

*   API
*   需要注意:直接使用无法切割中文 需要添加分词器 比如jieba分词

###### jieba分词的安装

    pip install jieba

    conda install jieba

    import jieba
    jieba.cut()   返回词语组成的生成器

<!---->

    from sklearn.feature_extraction.text import CountVectorizer
    # 特征提取-文本特征提取

    入参:
     * stop_words  无意义的词语 忽略分析
       stop_words=[]]

    返回参数:
    CountVectorizer.get_feature_names() 单词列表

需要用`toarray()`进行将sparse矩阵转换为array数组

*   案例 英文 直接使用 CountVectorizer

<!---->

    from sklearn.feature_extraction.text import CountVectorizer
    # 特征提取-文本特征提取

    ## 特征数据集
    x=["life is short,i like pyhton", "life is too long , i dislike pyhton"]
    stop_word=["long"]
    ## 实例化文本特征提取器 stop_words来控制忽略哪些字段[]
    ver=CountVectorizer(stop_words=stop_word)
    ## 调用fit_transform进行输入和转换
    res=ver.fit_transform(x)

    # 使用 get_feature_names_out 获取特征名称
    feature_names = ver.get_feature_names_out()
    print(feature_names)
    ## 返回数组
    print(res.toarray())

*   案例 中文 使用jieba分词
    注意 `" ".join(jieba.cut(text))` 这里是需要带空格的,不然拆分不正确

<!---->

    from sklearn.feature_extraction.text import CountVectorizer
    import jieba

    # 特征数据集
    data = ["我爱你python", "生活太长久,我用python"]

    # 分词处理方法
    def cut_word(text):
        # 使用 jieba 分词，并保留每个词之间的空格
        text = " ".join(jieba.cut(text))
        return text

    # 将原始数据转换为分好词的形式
    text_list = []
    for sent in data:
        text_list.append(cut_word(sent))

    # 定义停用词列表
    stop_word = ["太", "长久", "，"]

    # 实例化文本特征提取器，控制忽略哪些字段
    ver = CountVectorizer(stop_words=stop_word)

    # 调用 fit_transform 进行输入和转换
    res = ver.fit_transform(text_list)

    # 使用 get_feature_names_out 获取特征名称
    feature_names = ver.get_feature_names_out()
    print(feature_names)

    # 返回数组
    print(res.toarray())

*   案例 CountVectorizer+TfidfTransformer 计算文字出现的比例

<!---->

    from sklearn.feature_extraction.text import CountVectorizer,TfidfTransformer
    import jieba
    # 特征提取-文本特征提取 使用jieba分词

    ## 特征数据集

    def cut_word(text):
        text=" ".join(list(jieba.cut(text)))
        return text

    ##将原始数据转换为分好词的形式
    text_list=[]
    for sent in data:
        text_list.append(cut_word(sent))

    ## 实例化文本特征提取器 top_words来控制忽略哪些字段[]
    stop_word = []
    ver=CountVectorizer(stop_words=stop_word)
    ## 调用fit_transform进行输入和转换
    res=ver.fit_transform(text_list)
    # 使用 get_feature_names_out 获取特征名称
    feature_names = ver.get_feature_names_out()
    print(feature_names)
    ## 返回数组
    print(res.toarray())

    # 实例化 TF-IDF 转换器
    tfidf_transformer = TfidfTransformer()
    # 调用 fit_transform 进行输入和转换
    tfidf = tfidf_transformer.fit_transform(res)
    # 打印特征名称
    print("Feature Names:", feature_names)
    ## 返回数组
    print(tfidf.toarray())

#### 2.4 单一文本提取 ->LabelEncoder

主要用于将类别标签（通常是非数值类型的字符串）转换为整数。它将每个不同的标签映射到一个唯一的整数，通常是从 0 开始编号。

```
from sklearn.preprocessing import LabelEncoder

# 创建 LabelEncoder 实例
le = LabelEncoder()

# 假设我们有一些类别标签
labels = ['apple', 'banana', 'apple', 'orange', 'banana', 'banana']

# 将类别标签转换为数值标签
numeric_labels = le.fit_transform(labels)

print("Original labels:", labels)
print("Numeric labels:", numeric_labels)
print("Class mapping:", dict(zip(le.classes_, le.transform(le.classes_))))

# 将数字标签还原到原始标签
le.inverse_fit_transform(numeric_labels)

```
