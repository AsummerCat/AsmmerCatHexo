# 常用的idea插件

1. Codota  :代码自动补全

2. Mybatis-log-plugin :查看mybatis日志

3. Free-idea-mybatis :mybatis对应mapper跳转  

<!--more-->

4. CodeGlance  : 右侧缩略图

5. RestfulToolkit : 
根据 URL 直接跳转到对应的方法定义 ( Ctrl \ or Ctrl Alt N );

6. Alibaba Java Coding Guidelines: 阿里代码规范

7. lombok :免写set get

8. Nyan Progress Bar :彩虹猫进度条
