# 线性回归

## 1.定义

利用回归方程(函数) 对一个或者多个自变量(特征值)和因变量(目标值)之间关系进行建模的一种分析方式

*   特点: 只有一个自变量的情况称为单变量回归,多余一个自变量情况的叫做多元回归

*   使用场景: 放假预测 销售额度预测 贷款额度预测

#### 1.1 线性回归的特性

一种是`线性关系`

一种是`非线性关系`

## 2.线性回归API的使用

`sklearn.linear_model.LinearRegression()`
参数:

*   LinerRegressionn.coef\_: 回归系数

```
# 导入线性回归模块
from sklearn.linear_model import LinearRegression 

```

#### 2.1 案例
```
    # 导入线性回归模块 
    from sklearn.linear_model import LinearRegression 

    ## 构建数据集
    #### 平时成绩,期末成绩
    x=[[80,86],[82,80],[85,78],[90,90],[86,82],[82,90],[78,80]]
    #### 最终成绩
    y=[84.2,80.6,80.1,90,83.2,87.6,79.4]

    ## 模型训练
    #### 实例化模型 (正规方程)
    estimator=LinearRegression()

    ## 使用fit方法进行训练
    estimator.fit(x,y)

    #### 回归系数
    estimator.coef_
    print("数据集的特征的回归系数:",estimator.coef_)

    ## 模型预测
    ####  输出预测值
    res=estimator.predict([[100,80]])
    print("输出平时成绩100分,期末成绩80的最终成绩:",res)
    #### 输出预测准确率 
    estimator.score(x,y)
```
## 3.线性回归中的损失和优化算法

*   线性回归中的损失函数
    `预测的线性回归线 可能跟真实数据有误差 误差部分则为损失`

`正规方程`和`梯度下降法` 都是用来优化模型使损失最小

*   使用正规方程对损失函数优化

```
求导,
优点:使用矩阵的方式直接求导最好的结果
缺点:当特征值过多过复杂时,求解速度太慢并且得不到结果

```

*   使用梯度下降法对损失函数优化
    是一个可微分的函数 , 可以理解为下山的过程(不停试探)
    在看不见下面的情况下,利用周围的信息来找到下山的路径,重复采用同一个方法,下山(也就是找到函数的最小值)

有一个(重要)步长的值 ->根据步长来控制每一步的距离,以保证 不要走太快(错过最低点),不要走太慢(时间被延长)

*   梯度下降和正规方程的对比
    \| 梯度下降 | 正规方程 |
    \| --- | --- |
    \| 需要选择学习率(步长) | 不需要 |
    \| 需要迭代求解 | 一次运算得出 |
    \| 特征数量较大可以使用 | 需要计算方,时间复杂度高O(n3) |

*   算法选择依据:
    \*\* 小规模数据:
    正规方程: LinerRegression(不能解决拟合问题)
    岭回归

\*\* 大规模数据:
梯度下降法: SGDregressor

#### 3.1 梯度下降算法的变种

*   全梯度下降算法(FGD)
    `使用全部的样本进行一个迭代更新`

*   随机梯度下降算法(SGC)
    `随机拿出一个样本进行迭代 ,下次再拿出一个`

*   小批量梯度下降算法(mini-bantch)
    `其中拿出部分样本进行迭代`

*   随机平均梯度下降算法(SAG)
    `每次拿出来后,保留一个梯度,下次取前面的所有梯度的平均值取进行一个更新`
    SAG 的核心思想是在每次迭代中，不仅使用当前批次的数据来计算梯度，还利用过去所有批次的梯度信息来更新参数。具体来说，SAG 保存了所有训练样本的梯度估计，并在每次迭代时更新这些估计。

## 5.梯度下降API的使用 SGDRegressor

SGDRegressor函数:
实现了随机梯度下降学习,支持不同的`loss函数`和`正则化惩罚项`来拟合线性回归模型

```
sklearn.linear_model.SGDRegressor(loss="squared_loss",fit_intercept=True,
learing_rate='invscaling',etaO=0.01)
 
参数:
  * loss:损失类型  
       ** loss="squared_loss": 普通最小二乘法
  * max_iter  最大迭代次数  1000       
  
  * fit_intercept: 是否计算偏置 
  * learing_rate: 学习率填充

属性:
  * SGDRegressor.coef_: 回归系数
  * SGDRegressor.intercept: 偏置
  
  
```

##### 5.1 随机梯度下降算法实现 SGDRegressor

需要注意的是 需要对数据进行标准化操作

```
from sklearn.linear_model import LinearRegression,SGDRegressor
from sklearn.preprocessing import StandardScaler

# 构建数据集
x = [[80, 86], [82, 80], [85, 78], [90, 90], [86, 82], [82, 90], [78, 80]]
y = [84.2, 80.6, 80.1, 90, 83.2, 87.6, 79.4]

# 模型训练
# estimator = SGDRegressor(max_iter=1000)
scaler = StandardScaler()
x_scaled = scaler.fit_transform(x)


estimator = SGDRegressor(max_iter=1000, tol=1e-3, penalty='l2', learning_rate='invscaling', eta0=0.01, random_state=0)
# 使用fit方法进行训练
estimator.fit(x_scaled, y)

# 输出回归系数和偏置
print("数据集的特征的回归系数:", estimator.coef_)
print("数据集的特征的偏置:", estimator.intercept_)

# 模型预测
## 对新数据点进行相同的标准化处理
new_data_scaled = scaler.transform([[100, 80]])
res = estimator.predict(new_data_scaled)
print("输出平时成绩100分,期末成绩80的最终成绩:", res)

```

## 6.正则化

#### 6.1 岭回归 算法 (常用)

岭回归是线性回归的正则化版本 , 在原来的线性回归的`cost function`中添加了正则项

*   API

```
sklearn.linear_model.Ridge(alpha=1.0,fit_intercept=True,solver="auto",normalize=False)
* 具有L2正则化的线性回归
参数:
* alpha:正则化力度  取值: 0~1 1~10 值越小,惩罚力度越小
* solver:会根据数据自动选择优化算法   sag: 如果数据集,特征都比较大,选择该随机梯度下降优化
* normalize:数据是否进行标准化 
   normalize=False;可以在fit之前调用preprocessing.StandardScaler标准化数据

```

`Ridge`方法相当于`SGDRegressor( penalty='l2',loss="squared_loss" )`,只不过`SGDRegressor`实现了一个普通的随机梯度下降学习,
推荐使用`Ridge`(实现了SAG 随机梯度下降算法)

\*\* 这里封装了一个交叉校验的方法

```
sklearn.linear_model.RidgeCV(_BaseRidgeCV,RegressorMixin)
* 具有L2正则化的线性回归,可以进行交叉验证 

**RegressorMixin 可以传入Ridge=(0.1,1.0,10.0), 之类的超参数

```
