# 2.机器学习方法 LLM

### 2.1 机器学习 Machine Learning

自行从数据中学习和改进

### 2.2 监督学习 Supervised Learning

输入数据集->模型训练(监督员 需要大量人工介入标注)打标签 ->机器学习算法 ->模型(测试数据集<也就是我们要输入的内容>)->预测(输出)

### 2.3 监督学习应用 分类 Classification

进行识别(比如猫图片/狗图片 打标签) ->识别是狗还是猫

### 2.4 监督学习应用 回归 Regression

比如根据相似度-> 地段/距离 进行(预测) 房价

### 2.5 无监督学习 Unsupervised Learning

原始数据(无标签)->机器学习算法(寻找规则,特征)->模型->输出

### 2.6 无监督学习应用  聚类 Clustering

可以这么理解:  基于形状分类 ,基于颜色分类 ,根据不同维度进行分类聚合

### 2.7 无监督学习应用  规则关联 Association Rule

比如: 推荐算法\~ 从无序的数据中筛选\
比如购物车->买了红酒 ->匹配规则 ->推荐 红酒杯

### 2.8 强化学习 Reinforcement Learning

通过环境的反馈进行学习

模型训练过程中
根据你的动作来识别是否正确 (比如回答正确就继续按照这个方向,回答错误下次就不走这个方向)
例如:阿尔法狗 ,比如智能驾驶(选择最合理最优线路)

# 3. 机器学习效果评估(拟合)

拟合->用来评估机器学习反馈效果的

欠拟合(Under-fitting):
结果不符合预期 (数据集质量不够,数据集量不够)

最佳拟合(Optimal-fitting):
输入的内容 符合预期

过拟合(Over-fitting):
训练成果 只能根据训练集来匹配 无法泛化 (如果输入训练集之外的数据 可能会识别不出来)
