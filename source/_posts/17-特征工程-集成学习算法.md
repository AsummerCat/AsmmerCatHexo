# 集成学习

集成学习是通过建立几个模型来解决单一预测问题.
它的工作原理是`生成多个分类器/模型`,各自独立的学习和作出预测.
`这些预测最后结合成组合预测,因此由于任何一个单分类的做出预测`

只要单分类器的表现不太差,集成学习的结果总是要好于单分类器的

## 1.集中学习中的`Boosting`和`Bagging`

Boosting (会经常用到)

## 1.0 `boosting`和`Bagging`  区别

*   区别一: 数据方面
    \*\* Bagging:  对数据进行采样驯良
    \*\* Boosting: 根据前一轮的学习结果调整数据的重要性

*   区别二: 投票方面
    \*\* Bagging:  所有学习器平权投票
    \*\* Boosting: 对学习器进行加权投票

*   区别三: 学习顺序
    \*\* Bagging:  学习是并行的,每个学习器没有依赖关系
    \*\* Boosting: 学习是串行的,学习有先后顺序

*   区别四: 主要作用
    \*\* Bagging:  主要用于提高泛化性能(解决过拟合,也可以说降低方差)
    \*\* Boosting: 主要是用于提高训练精度(解决欠拟合,也可以说是降低偏差)

<!--more-->

#### 1.1 Boosting

*   `Boosting` :
    主要用来处理:`欠拟合问题`
    弱弱组合`变强`
    主要方法: Boosting逐步增强学习

简而言之: `就是每加入一个弱学习器,整体能力就会得到提升`
代表算法:
`Adaboost`：通过调整样本权重来逐步增强学习器。
`GBDT`：通过梯度下降的方式逐步减小损失函数。
`XGBoost`：在 GBDT 的基础上增加了正则化项，提高了模型的泛化能力。
`LightGBM`：通过直方图算法和 GOSS/EFB 等技术，提高了训练速度和效率。
这些算法都是 Boosting 方法的典型代表，通过组合多个弱学习器来形成一个强学习器，从而提高模型的性能。

###### 1.1.2 Adaboost算法的API `AdaBoostClassifier` 自适应增强

*   完整流程
    1.训练第一个学习器：使用初始数据集训练第一个弱学习器。
    2.调整数据分布：根据第一个学习器的预测误差，调整数据集中每个样本的权重，使得错误预测的样本权重增加。
    3.训练第二个学习器：使用调整后的数据集训练第二个弱学习器。
    4.继续调整数据分布：根据第二个学习器的预测误差，再次调整数据集中每个样本的权重。
    5.依次训练学习器，调整数据分布：重复上述步骤，直到训练完所有弱学习器。
    6.加权投票（Adaboost）：最终的预测结果由所有弱学习器的加权投票决定。
    总结

    from sklearn.ensemble import AdaBoostClassifier
    from sklearn.datasets import make_classification
    from sklearn.model_selection import train_test_split

    # 生成分类数据集

    # 预测
    y_pred = adaboost_clf.predict(X_test)

    # 评估模型
    accuracy = accuracy_score(y_test, y_pred)

        colsample_bytree=1.0,         # 列采样比例，用于防止过拟合
        reg_alpha=0,                  # L1 正则化项的权重
        reg_lambda=1,                 # L2 正则化项的权重
        random_state=42               # 随机种子，保证结果可复现
    )

    # 训练模型
    xgb_clf.fit(X_train, y_train)

    # 预测
    y_pred = xgb_clf.predict(X_test)

    # 评估模型
    accuracy = accuracy_score(y_test, y_pred)
    print(f"Accuracy: {accuracy:.2f}")

#### 1.2 Bagging

*   `Bagging`:
    主要用来处理:`过拟合问题`
    互相遏制变`壮`
    主要方法: Bagging采样学习集成

*   `主要流程`
    1.采样不同数据集
    2.训练分类器 (多个)
    3.平权投票,获取最终结果

*   `Bagging`集成优点:
    `Bagging`+决策树/线性回归/逻辑回归/深度学习  ====`Bagging`集成学习算法
    经过上面方式的组成的集成学习方法:
    1.均可再原有算法上提高约2%左右的泛化正确率
    2.简单,方便,通用

###### 1.2.1 随机森林构造过程

在机器学习中,`随机森林是一个包含多个决策树的分类器`,并且其输出的类别是有个别树输出的众数而定.

随机森林=`Bagging`+决策树

例如: 你训练了5个树,其中有4个树的结果都是True,1个树的结果是False,那么最红投票结果就是True

*   主要构造过程:
    1.一次随机选出1个样本,有放回的抽样,重复N次(有可能出现重复的样本)
    2.随机去选出m个特征, m<\<M(特征数目),建立决策树

*   完整流程:
    1.随机选择m条数据
    2.随机抽取k个数据特征
    3.训练决策树(默认cart树) 基尼不纯度
    4.重复1-3步构建n个弱决策树
    5.平权投票集成n个弱决策树

*   包外估计(Out-of-Bag Estimate)

在随机森林构造过程中,如果进行有放回的抽样,我们会发现,总是有一部分样本我们选不到

    2.当基学习器是神经网络时,可使用包外样本来辅助早起停止以减少过拟合
