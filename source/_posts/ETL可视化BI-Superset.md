# ETL可视化BI-Superset
对于分析后的数据进行展示,类似大屏

其他的还有 powerBi等等
## 运行
```
任何目录下

superset run -h node1(或者ip地址) -p 8080 --with-threads --reload --debugger

终端关闭即停止
```
<!--more-->
## 地址:
```
http://ip:8080/superset/welcome
```
