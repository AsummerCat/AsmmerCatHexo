# 4.开始更新conda 并且基于python3.10 安装基础依赖
```

    确保环境正确
    conda update conda
    conda update --all

    conda activate AI-python-3-12-7

    使用pip安装包
    pip3 install -r requirement.txt
```

requirement.txt内容为
```

    matplotlib==3.7.2
    numpy==1.24.3
    pandas==2.0.3
    tables==3.8.1
    jupyter==3.2.0
```

或者使用`conda`来管理依赖

```

    conda install matplotlib==3.7.2
    conda install numpy==1.24.3
    conda install pandas==2.0.3
    conda install tables==3.8.1
    conda install pytables=3.8.1  # 注意：`tables` 库在 Conda 中通常称为 `pytables`
    conda install jupyter==3.2.0

    或者不指定版本
    conda install matplotlib
    conda install numpy
    conda install pandas
    conda install tables
    conda install pytables  # 注意：`tables` 库在 Conda 中通常称为 `pytables`
    conda install jupyter
```

## 再或者直接创建一个环境一次性安装这些依赖
```
    conda create --name new_env python=3.12 pandas numpy matplotlib tables jupyter
    conda activate new_env
```
