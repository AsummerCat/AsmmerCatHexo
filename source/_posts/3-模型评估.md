根据数据集的目标值不同,可以把模型评估分为`分类模型评估`,`回归模型评估`



# 1.回归模型评估

平方根误差 `RMSE` :
预测值 减 真实值的平方和除以样本数量,然后开根号
相对平方误差`RSE` :
(预测值 减 真实值的平方和) 除以 (真实值的均值 减 真实值的平方和)
平均绝对误差`MAE` :
预测值 减 真实值的绝对值的和,然后除以样本数量
相对绝对误差`RAE` :
(预测值 减 真实值的绝对值的和)除以(真实值的均值 减 真实值的绝对值的和)

误差越小越好

# 3.拟合

### 3.1欠拟合 under-fitting

`表现: 训练集中表现不好,测试集中表现也不好`
模型学习太过粗糙,训练集中的样本特征关系没有学出来

结果不符合预期 (数据集质量不够,数据集量不够)

### 3.2 过拟合  Over-fitting

`表现: 训练集中表现比较好,测试集中表现不好`
在训练样本中表现过于优越,导致在测试数据集中表现不佳

学习到的东西太多,学习到的特征多,不好泛化

训练成果 只能根据训练集来匹配 无法泛化 (如果输入训练集之外的数据 可能会识别不出来)

## 4.KD树

KD树指的是:
为了避免每次都重新计算一遍距离,算法会把距离信息保存在可以一棵树上, 这样计算之前从树里查询距离信息,尽量避免重新计算

其原理是:
如果A和B距离很远,B和C距离很近,那么A和C的距离也很远.有了这个信息,就可以在合适的时候跳过距离远的点

这样优化后的算法复杂度可降低到O(DNlog(N))

(结构类似二叉树)

## 5.模型调优-正则化  (过拟合处理方案)

`通过限制高次项的系数进行防止过拟合`

*   L1正则化
    作用: 可以使其中一些W的值直接为0,删除这个特征的影响
    比如: LASSO回归

*   L2正则化
    作用: 把高次项前面的系数变成特别小的值 ,也就是使其中一些W变小,都接近0,削弱某个特征的影响
    比如: 岭回归(Ridge)

## 6.模型调优方式

定位超参数处理

*   手动调优：适用于小规模数据集和少量超参数。

*   网格搜索(`GridSearchCV`)：适用于超参数数量较少且计算资源充足的情况。
    `from sklearn.model_selection import GridSearchCV`

*   随机搜索(`RandomizedSearchCV`)：适用于超参数数量较多且计算资源有限的情况。
    `from sklearn.model_selection import RandomizedSearchCV`

*   贝叶斯优化(`BayesSearchCV`)：适用于需要高效搜索最优解的情况，但实现复杂。
    `from skopt import BayesSearchCV`
